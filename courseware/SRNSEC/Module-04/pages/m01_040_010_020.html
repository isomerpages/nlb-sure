Lesson Outline

This lesson will cover the following:

  • Using skimming and scanning techniques to extract information from the news articles.
  • How to paraphrase the information obtained
  • Writing citations for newspaper articles
Learning Objectives

At the end of this lesson, you will be able to:

  • Extract information from newspaper articles using skimming and scanning techniques
  • Paraphrase information using the guidelines provided
  • Cite newspaper articles using the guidelines provided
